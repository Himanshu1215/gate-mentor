# Chapter 1: Vector Spaces and Subspaces

A **vector space** consists of a set of vectors, along with two operations: vector addition and scalar multiplication. These operations must satisfy ten axioms, including closure, associativity, commutativity, and the existence of a zero vector and additive inverses.

A **subspace** of a vector space is a subset that is itself a vector space under the same operations. To prove a subset is a subspace, we only need to check three conditions:
1. The zero vector is in the subset.
2. It is closed under vector addition (if u and v are in the subset, u + v is in the subset).
3. It is closed under scalar multiplication (if u is in the subset and c is a scalar, c*u is in the subset).

For example, a line through the origin in R^3 is a subspace. A line not through the origin is NOT a subspace because it does not contain the zero vector.
